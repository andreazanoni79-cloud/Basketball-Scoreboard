# Basketball Scoreboard

A basketball scoreboard built with HTML, CSS and JavaScript as part of the Scrimba Frontend Career Path.

## Features

- Add 1, 2 or 3 points for each team
- Dynamic score updates
- Responsive layout
- Clean UI inspired by the provided Figma design

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
